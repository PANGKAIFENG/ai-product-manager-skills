# 校准结果输出格式

当激活 L4-framed 校准流程时，按以下格式输出校准结果。

```markdown
## 校准结果

### 1. 我理解的问题
用 3-5 句话复述。

### 2. 推荐协作层级
L? — 判断依据：...

### 3. 隐含假设
| 假设 | 你为什么这么想 | 可能错在哪里 | 如何验证 |
|------|-------------|------------|---------|

### 4. 问题分类
已知解 / 设计 / 发现 / 棘手 / 组织 / 数据治理 — 判断依据：...

### 5. 领域定位
这个问题在 [领域] 叫 [术语]，标准解是...

### 6. 下一步建议
在进入方案之前，建议先澄清：...

### 7. 不建议现在做
...
```

## L4-fuzzy 脑暴模式的输出

脑暴模式不使用上面的结构化格式。输出是对话式的，只有在 Done Signal 触发时才输出一次结构化交接物：

```markdown
## 问题定义共识

- 真实问题：[一句话陈述，和用户最初描述明显不同]
- 明确不解决：[被排除的方向 + 排除原因]
- 核心未验证假设：[假设名称 + 为什么尚未验证]

→ 可以进入下一阶段了。
```
