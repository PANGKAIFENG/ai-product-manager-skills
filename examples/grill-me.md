# Example: grill-me

## Use When

You have a plan or product direction and want a focused pressure test.

## Prompt

```text
$grill-me

我的方案是：先做一个 AI PM Skill 库，覆盖脑暴、研究、PRD、评审和方案压测，
通过 GitHub 开源推广，然后再做团队分发。
请拷问这个方案，重点问最可能失败的地方。
```

## Expected Output Shape

- Restates the plan.
- Identifies major assumptions and failure modes.
- Asks one focused question at a time.
- Gives a recommended answer or current assumption for each question.
- Pushes toward a clearer decision tree rather than a long generic list.

## Good Follow-Up

```text
基于刚才的回答，帮我收敛成一个最小可行推广计划。
```
