# 中文短帖

我整理了一个开源项目：AI Product Manager Skills Library。

它是给 Codex / Claude Code 用的中文优先 AI PM Skill 库，补的是 Superpowers 之前的产品侧工作流：

脑暴校准 -> 主题研究 -> 技术调研 -> PRD 起草 -> PRD 评审 -> 方案压测 -> 再交给 Superpowers 做开发计划。

目前包含 6 个 Skill：

- `ai-collaboration-calibration`
- `research-topic-compiler`
- `tech-research`
- `prd-architect`
- `prd-review`
- `grill-me`

适合产品经理、产品负责人、AI PM workflow 探索者。
