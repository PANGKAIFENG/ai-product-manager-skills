# 中文短帖

我整理了一个开源项目：AI Product Manager Skills Library。

它是给 Codex / Claude Code 用的中文优先 AI PM Skill 库，补的是 Superpowers 之前的产品侧工作流：

脑暴校准 -> 主题研究 -> 决策调研 -> PRD 起草 -> PRD 评审 -> PRD 拆 issue -> UI mockup / 方案压测 -> 再交给 Superpowers 做开发计划。

目前包含 9 个 Skill：

- `ai-collaboration-calibration`
- `research-topic-compiler`
- `decision-research`
- `prd-architect`
- `prd-review`
- `prd-to-issues`
- `ui-mockup-desktop-workbench`
- `grill-me`
- `ai-work-assetization-diagnoser`

适合产品经理、产品负责人、AI PM workflow 探索者。
