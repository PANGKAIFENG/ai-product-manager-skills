# English Project Summary

AI Product Manager Skills Library is a Chinese-first Agent Skills repository for product managers using Codex, Claude Code, or similar agentic coding tools.

It complements Superpowers by covering the product-side workflow before engineering execution:

- product problem calibration
- complex exploration and reusable asset extraction
- product research, competitive analysis, and decision-driven research
- design brainstorming before PRD or implementation planning
- PRD drafting
- PRD review
- PRD-to-issue backlog handoff
- UI wireframe and high-fidelity desktop UI handoff
- plan pressure testing
- AI work assetization diagnosis
- requirements handoff

The public library currently includes thirteen Skills:

- `ai-collaboration-calibration`
- `complex-exploration`
- `research-topic-compiler`
- `competitive-analysis`
- `decision-research`
- `brainstorming`
- `prd-architect`
- `prd-review`
- `prd-to-issues`
- `ui-wireframe-to-html`
- `ui-mockup-desktop-workbench`
- `grill-me`
- `ai-work-assetization-diagnoser`
